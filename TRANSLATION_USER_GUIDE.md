# ModelScope翻译功能使用指南

## 功能概述

AI图像生成平台现在集成了ModelScope翻译功能，可以将中文提示词翻译成英文，适配需要英文输入的AI图像生成模型。

## 🎯 翻译支持

### 支持的模型：
- ✅ **ChatGPT-5模型** - 提示词翻译
- ✅ **Nano Banana模型** - 提示词翻译  
- ✅ **ModelScope模型** - 正面提示词翻译
- ✅ **ModelScope模型** - 负面提示词翻译

### 翻译方向：
- **单向翻译：** 中文 → 英文
- **适用场景：** 图像生成提示词翻译

## 🚀 使用方法

### 方法一：翻译按钮
1. 在任何提示词输入框旁边找到 **"翻译"** 按钮
2. 在输入框中输入中文提示词
3. 点击 **"翻译"** 按钮
4. 系统自动将中文翻译为英文并替换原文本

### 方法二：三空格触发
1. 在提示词输入框中输入中文文本
2. 在文本末尾输入 **三个连续空格**：`   `
3. 系统自动移除这三个空格并触发翻译
4. 翻译结果直接替换原文本

## 📝 使用示例

### 场景一：基础提示词翻译
```
输入：一只可爱的小猫在花园里玩耍
输出：a cute kitten playing in a garden
```

```
输出：a young woman in a red dress walking in a garden with bright sunlight and blooming flowers
```

### 场景三：艺术风格描述
```
输入：印象派风格的日出风景画，色彩鲜艳，笔触流畅
输出：impressionist style sunrise landscape painting with vivid colors and smooth brushstrokes
```

## ⚙️ 技术规格

### 翻译引擎：
- **模型：** iic/nlp_imt_translation_zh2en
- **提供商：** ModelScope (阿里巴巴达摩院)
- **精度：** 专门优化的中文→英文翻译
- **速度：** 平均响应时间 < 1秒

### API配置：
- **密钥：** MODELSCOPE_API_KEY（已配置）
- **接口：** https://api-inference.modelscope.cn/
- **认证：** Bearer Token

## 🔧 故障排除

### 常见问题：

**Q: 翻译按钮点击无反应？**
A: 检查网络连接，确保ModelScope API服务正常

**Q: 翻译结果为空？**
A: 请先在输入框中输入要翻译的中文文本

**Q: 三空格触发不工作？**
A: 确保在文本末尾输入完整的三个空格字符

**Q: 翻译质量不理想？**
A: ModelScope翻译专门优化了提示词翻译，比通用翻译更准确

### 状态提示：
- **"翻译中..."** - 正在处理翻译请求
- **"翻译成功"** - 翻译完成并替换了原文本
- **"翻译失败"** - 出现错误，请检查网络连接

## 🎨 最佳实践

### 提示词翻译建议：
1. **保持关键词完整性** - 确保重要的描述词汇准确翻译
2. **分句处理** - 长文本可以分段翻译再合并
3. **风格词汇** - 艺术风格、色彩描述等词汇翻译效果特别好
4. **避免歧义** - 复杂的多义词可能需要人工调整
